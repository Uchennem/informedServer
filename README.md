# informedServer

`informedServer` is a student-focused social connection platform designed for students on college campuses.

## 🚀 What it is
A backend service that powers a community platform for campus-based student connections, events, clubs, and discussions. It aims to help students easily find, join, and interact with peers who share interests and campus affiliations.

## 🧩 Key Features
- User authentication and profiles
- Campus and club discovery
- Event creation and RSVP
- Messaging and discussion threads

## 🛠️ Getting Started
1. Clone the repo:
   ```bash
   git clone <repo-url>
   cd informedServer
   ```
2. Install dependencies:
   ```bash
   # (example for Node.js projects)
   npm install
   ```
3. Start the server:
   ```bash
   npm start
   ```

## 🧪 Running Tests
```bash
npm test
```
